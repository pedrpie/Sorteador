# Sorteador

Projeto desenvolvido em **React** com o objetivo de sortear **nomes ou números** de forma simples, rápida e intuitiva.

O site permite que o usuário adicione nomes, defina intervalos numéricos e realize sorteios com apenas um clique, tudo em uma interface limpa e responsiva.

---

## Funcionalidades

- Adicionar e listar nomes
- Sortear um nome aleatoriamente
- Sortear um número dentro de um intervalo definido
- Validações de entrada para evitar erros
- Interface moderna com Tailwind CSS

---

## Tecnologias utilizadas

- **React**
- **Vite**
- **JavaScript**
- **Tailwind CSS**
